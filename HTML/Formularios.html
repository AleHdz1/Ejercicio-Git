Formularios en HTML

El objetivo de los formularios en HTML es crear la comunicacón o interactividad con mi usuario final

  • input: Se usa para crear controles interactivoscon la foinalidad de recibir datos del usuario

      • Atributos

    1. type: Tipo de información que quieres recibir para poder interpretarla como tal. Pj. Text,number,password,email,etc.
    2. Placeholder: Podemos guiar al usuario escribiendo la instrucción o la info que tiene que poner.
    3. Maxlength: largo máximo de caracteres que va a admitir el input
    4. Required: Campos obligatorios que de no ser llenados no permitirá enviar el formulario al campo que se le asigne volviendo al input faltante.Validación sencilla.
    5. Name: Para identificar los input en los archivos como php
    6. ID: Sirve para darle un valor como identificador único
    7. For: me da pie a que comience a escribir en el input seleccionado
