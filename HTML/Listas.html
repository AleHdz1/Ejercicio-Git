Listas HTML

Las listas HTML me permitirán crear un conjunto de elementos en forma de lista dentro de nuestro sitio web, todos irán precedidos generalmente por una viñeta o un número.


Listas ordenadas

Son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos lo elementos numerados. Es decir, cada uno de los elementos irán precediendo de un número o letra que establezca su orden.

Lenguajes que aprenderemos

  1. HTML
  2. CSS
  3. JS

Listas desordenadas

Las listas desordenadas me sirven para mostrar los elementos sin ningún tipo de orden, simplemente precedidos por una viñeta que puede ser un punto, cuadrado, etc.
